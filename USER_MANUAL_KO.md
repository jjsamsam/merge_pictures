# Merge Picture 사용자 안내서

## 1. 프로그램 소개

`Merge Picture`는 여러 장의 사진 파일과 PDF 파일을 하나의 PDF 문서로 합치는 데스크톱 프로그램입니다.

지원 입력 형식:
- JPG / JPEG
- PNG
- WEBP
- PDF

지원 운영체제:
- macOS
- Windows

이 프로그램으로 할 수 있는 일:
- 사진 여러 장을 하나의 PDF로 만들기
- PDF 여러 개를 하나로 합치기
- 사진과 PDF를 섞어서 하나의 PDF로 만들기
- 파일 순서를 바꿔 원하는 순서대로 문서 만들기

## 2. 설치 방법

### macOS

배포 파일 예시:
- `Merge Picture.app`
- `Merge Picture_0.1.0_aarch64.dmg`

설치 방법:
1. `.dmg` 파일을 엽니다.
2. `Merge Picture.app`를 `Applications` 폴더로 드래그합니다.
3. 설치가 끝나면 `Applications`에서 실행합니다.

### Windows

배포 파일 예시:
- 설치형 `.exe` 또는 `.msi`

설치 방법:
1. 설치 파일을 실행합니다.
2. 화면 안내에 따라 설치를 완료합니다.
3. 설치 후 시작 메뉴 또는 바탕화면 바로가기에서 실행합니다.

참고:
- Windows에서는 WebView2가 필요할 수 있습니다.
- 설치 프로그램이 필요한 구성요소를 자동으로 설치하거나 안내할 수 있습니다.

## 3. 첫 실행 시 안내

처음 실행하면 메인 화면에서 다음 요소를 볼 수 있습니다.

- `Add Files`
  - 파일 선택 창으로 사진/PDF를 추가합니다.
- 드래그 앤 드롭 영역
  - 파일을 창에 끌어다 놓아 추가할 수 있습니다.
- `Merge Queue`
  - 현재 병합할 파일 목록이 표시됩니다.
- `Output`
  - 저장할 파일 이름과 이미지 페이지 옵션을 설정합니다.
- `Merge To PDF`
  - 최종 PDF를 생성합니다.

## 4. 기본 사용 방법

### 방법 1: 파일 선택으로 추가

1. `Add Files` 버튼을 클릭합니다.
2. 병합할 사진 또는 PDF 파일을 선택합니다.
3. 선택한 파일이 `Merge Queue`에 추가됩니다.

### 방법 2: 드래그 앤 드롭으로 추가

1. Finder 또는 파일 탐색기에서 파일을 선택합니다.
2. 프로그램 창 위로 끌어옵니다.
3. 드롭 영역이 강조되면 놓습니다.
4. 파일이 `Merge Queue`에 추가됩니다.

## 5. 파일 순서 바꾸기

병합 결과는 `Merge Queue`의 위에서 아래 순서대로 만들어집니다.

순서 변경 방법:
- `↑` 버튼: 해당 파일을 위로 이동
- `↓` 버튼: 해당 파일을 아래로 이동

파일 삭제 방법:
- `×` 버튼을 눌러 해당 파일을 목록에서 제거

전체 삭제 방법:
- `Clear All` 버튼을 눌러 목록 전체 제거

## 6. 미리보기 정보 읽는 법

각 파일 카드에는 다음 정보가 표시됩니다.

### 이미지 파일
- 썸네일 미리보기
- 해상도 예시: `1600×900px image`
- 파일 크기 예시: `1.4 MB`

### PDF 파일
- 문서 형태의 카드 미리보기
- 페이지 수 예시: `3 pages`
- 파일 크기 예시: `218 KB`

## 7. 출력 옵션 설정

오른쪽 `Output` 영역에서 저장 관련 옵션을 설정할 수 있습니다.

### Output file name

저장될 PDF 파일 이름입니다.

예:
- `merged.pdf`
- `견적서_통합본.pdf`

확장자 `.pdf`를 입력하지 않아도 자동으로 붙습니다.

### Image Page Layout

이 옵션은 `이미지 파일`에만 적용됩니다.
이미 원래 PDF인 파일에는 적용되지 않습니다.

#### Page size

- `Auto`
  - 이미지 크기에 맞춰 페이지 크기를 자동으로 만듭니다.
- `A4`
  - A4 용지 크기에 맞춰 이미지를 배치합니다.
- `Letter`
  - Letter 용지 크기에 맞춰 이미지를 배치합니다.

#### Margins

- 이미지 페이지 바깥 여백을 조절합니다.
- 값이 클수록 이미지 주변 여백이 넓어집니다.

#### Image fit

- `Contain`
  - 이미지 전체가 페이지 안에 보이도록 맞춥니다.
  - 잘림이 적고 일반적으로 가장 무난합니다.
- `Cover`
  - 페이지를 꽉 채우도록 확대합니다.
  - 일부 가장자리가 잘릴 수 있습니다.
- `Fill`
  - 페이지 영역에 맞게 가로/세로를 각각 늘립니다.
  - 비율이 변형될 수 있습니다.

## 8. PDF로 저장하기

1. 파일을 모두 추가합니다.
2. 필요하면 순서를 조정합니다.
3. 출력 파일 이름과 이미지 옵션을 설정합니다.
4. `Merge To PDF` 버튼을 클릭합니다.
5. 저장 위치를 선택합니다.
6. 저장이 완료되면 상태 메시지에 결과가 표시됩니다.

## 9. 권장 사용 예시

### 예시 1: 사진 여러 장을 하나의 문서로 만들기

추천 설정:
- Page size: `A4`
- Margins: `10~15 mm`
- Image fit: `Contain`

이런 경우에 적합합니다:
- 스캔 사진 묶기
- 보고용 이미지 모음 만들기
- 제출용 PDF 만들기

### 예시 2: 여러 PDF를 하나로 합치기

추천 설정:
- 이미지 옵션은 크게 신경 쓰지 않아도 됩니다.

이런 경우에 적합합니다:
- 계약서 묶음
- 여러 장의 보고서 정리
- 발주서/견적서 통합

### 예시 3: 사진과 PDF를 함께 섞어 하나로 만들기

추천 설정:
- 이미지가 많이 포함되면 `A4 + Contain` 조합 권장

이런 경우에 적합합니다:
- 표지 이미지 + 본문 PDF
- 증빙 이미지 + 설명 PDF

## 10. 자주 묻는 문제

### 1. `Merge To PDF` 버튼이 비활성화되어 있습니다

가능한 원인:
- 파일이 하나도 추가되지 않음
- 브라우저 데모 import로만 추가됨

해결 방법:
- `Add Files` 또는 드래그 앤 드롭으로 다시 추가하세요.
- 실제 데스크톱 파일 경로가 들어가야 병합이 가능합니다.

### 2. 파일을 추가했는데 병합이 되지 않습니다

가능한 원인:
- 지원하지 않는 파일 형식
- 파일이 삭제되었거나 이동됨
- 손상된 PDF

해결 방법:
- JPG, PNG, WEBP, PDF인지 확인하세요.
- 파일이 원래 위치에 있는지 확인하세요.
- 손상된 PDF가 아닌지 확인하세요.

### 3. 저장이 되지 않습니다

가능한 원인:
- 저장 권한이 없는 폴더 선택
- 같은 파일을 다른 프로그램이 사용 중

해결 방법:
- 바탕화면이나 문서 폴더에 저장해 보세요.
- 이미 열려 있는 PDF 뷰어를 닫고 다시 시도해 보세요.

### 4. 이미지가 잘려 보입니다

가능한 원인:
- `Image fit`이 `Cover`로 설정됨

해결 방법:
- `Contain`으로 바꾸면 이미지 전체가 보이도록 맞춰집니다.

### 5. 이미지가 페이지에서 너무 작게 보입니다

가능한 원인:
- 여백이 너무 큼
- `Auto` 대신 고정 페이지 크기 사용 중

해결 방법:
- 여백을 줄이세요.
- 필요하면 `Auto` 페이지 크기를 사용해 보세요.

## 11. 제거(삭제) 방법

### macOS

1. `Applications` 폴더를 엽니다.
2. `Merge Picture.app`를 휴지통으로 이동합니다.
3. 필요하면 다운로드한 `.dmg` 파일도 삭제합니다.

### Windows

1. `설정 > 앱` 또는 `제어판 > 프로그램 제거`로 이동합니다.
2. `Merge Picture`를 선택합니다.
3. `제거(Uninstall)`를 실행합니다.

## 12. 사용 팁

- 사진만 합칠 때는 `A4 + Contain`이 가장 안정적입니다.
- 결과 순서가 중요하면 병합 전에 `Merge Queue` 순서를 꼭 확인하세요.
- 긴 문서 작업 전에는 먼저 2~3개 파일로 시험 저장해 보면 좋습니다.
- PDF 원본은 그대로 보관하고, 병합 결과는 새 파일로 저장하는 것을 권장합니다.

## 13. 빠른 시작 요약

1. 프로그램 실행
2. `Add Files` 또는 드래그 앤 드롭으로 파일 추가
3. 순서 조정
4. 출력 이름과 이미지 옵션 설정
5. `Merge To PDF`
6. 저장 위치 선택
7. 완료
